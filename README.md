# roguelite

Wannabe fushigi clone.

## Getting started

Prerequesites are:
- Node

Install dependencies with `npm install`

## Development server

Run test server with `npm start`

## Building

Generate a build with `npm run build`. Deployable build artifacts will be output to /dist.
